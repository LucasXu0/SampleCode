# -API-
        这是一个由百度音乐API获取数据的简易demo
        其中文件XKMusicSource这个model是用来解析JSON数据的，传入搜索的歌名，会返回歌曲信息：歌曲ID，歌手名字，歌曲名，下载地址，歌曲图片等。
        其中文件XKMusicDownload这个model是用来下载音乐，传入歌曲下载地址，其属性songURL会记录下载音乐的本地路径，传入播放器即可播放。
